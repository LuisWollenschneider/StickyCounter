# StickyCounter

> [!NOTE]
This App was _Vibe-Coded_ 🙃

A lightweight macOS app for simple counters — with support for floating (always-on-top) windows and detachable popout panels.

## Features

- **Named counters** — create, increment, decrement, variable step size
- **Sticky / floating mode** — pin the main window or any popout to float above all other windows (`Cmd + F`)
- **Popout panels** — detach individual counters into their own resizable floating windows
- **Folders** — organize counters into folders
